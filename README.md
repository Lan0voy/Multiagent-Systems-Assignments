## Структура робіт

### Завдання 1. Стратегічні ігри

Реалізовано програму мовою C++, яка аналізує три стратегічні ігри:

- дилема ув’язненого;
- гра в монетку;
- гра на перехресті.

Програма виконує пошук строго домінованих стратегій та чистих рівноваг Неша.

Файл програми:

    Завдання_1_multiagGames/multiagGames.cpp

Компіляція:

    g++ Завдання_1_multiagGames/multiagGames.cpp -o multiagGames

Запуск:

    ./multiagGames

У Windows:

    multiagGames.exe

### Завдання 2. PettingZoo TicTacToe

Розглянуто бібліотеку PettingZoo для моделювання мультиагентних середовищ.  
Для практичного експерименту використано середовище `tictactoe_v3`.

У програмі два агенти, `player_1` та `player_2`, по черзі роблять ходи на полі 3x3.  
Ходи обираються випадково, але тільки серед дозволених дій через `action_mask`.

Файл програми:

    pettingzoo_tictactoe.py

Встановлення залежностей:

    python -m pip install "pettingzoo[classic]"

Запуск без візуалізації:

    python pettingzoo_tictactoe.py --episodes 100

Запуск однієї гри з візуальним вікном:

    python pettingzoo_tictactoe.py --episodes 1 --render

### Завдання 3. Ray / RLlib

Розглянуто платформу Ray та підхід RLlib до організації мультиагентних середовищ.

Практична частина реалізує гру **«Камінь, папір, ножиці»** для двох агентів.  
Кожен агент випадково обирає одну з трьох дій. Один епізод складається з кількох раундів.

Файл програми:

    ray_multiagent_rps.py

Встановлення Ray:

    python -m pip install -U "ray[rllib]"

Локальний запуск:

    python ray_multiagent_rps.py --mode local --episodes 10 --rounds 5

Локальний запуск з деталями першого епізоду:

    python ray_multiagent_rps.py --mode local --episodes 10 --rounds 5 --details

Запуск через Ray:

    python ray_multiagent_rps.py --mode ray --episodes 100 --rounds 10
